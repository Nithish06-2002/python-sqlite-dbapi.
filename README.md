# Python SQLite DB-API Project

## Description
This project demonstrates how to use Python DB-API with SQLite to perform
basic database operations and retrieve data into a Pandas DataFrame.

## Concepts Covered
- SQLite database connection using sqlite3
- Cursor operations (execute, fetchmany, fetchall)
- SQL operations: CREATE, INSERT, SELECT, UPDATE
- Loading SQL data into Pandas using read_sql_query
